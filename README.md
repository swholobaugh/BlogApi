# My First Node App

## Installation Dependencies

```bash
yarn
```

## Enviromental Variables

Copy enviromental file and change it as needed

```bash
cp env.example .env
```

## Run Dev Server

```bash
yarn dev
```

## Build for Production

```bash
yarn build
```

## Run in production

```bash
yarn start

#Or if you are using PM2
pm2 start npm -- start
```
